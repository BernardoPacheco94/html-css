Principais formatações

Negrito

Aqui há um termo em negrito usando a tag <b> (não semantica/de forma)

Nesta frase há um termo em destaque usando a tag <strong>

Italico

Aqui está uma frase com aplicação de itálico usando a tag <i>

Nesta frase temos um termo em destaque usando a tag <em>

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Para definir a cor do mark no código todo, use a tag <style> em <head>, com o style como parâmetro

Texto Grande e Pequeno

Estamos criando um texto grande e outro pequeno usando as tags <big> (obsoleta) e <small> (válida)

Texto Deletado

Podemos marcar um texto como excluído, para indicar que ele pode ser lido mas deve ser desconsiderado, usando a tag <del>

Texto Inserido

Podemos marcar um texto como inserido, para dar ênfase e indicar que ele foi adicionado depois com a tag <ins>. Também existe a tag <u>, porém é não semantica

Texto Sobrescrito

Para inserir textos como x20-3, usar a tag <sup>

Texto Subescrito

Inserir textos como H2O, use a tag <sub>

Citações

Como diria um amigo: o computador é um burro muiro rápido. Utilizada a tag <q> de Quotation.

Para citações completas (mais longas) é usada a tag <blockquote>, como a citação do trecho abaixo: 

        

        Regateei com a vida por um centavo,
        E a Vida não quis pagar mais,

        Quando contava meus magros haveres.
        Pois a Vida é apenas empregador,
        Dá-lhe o que você pede,
        Mas desde que tenha estabelecido o salário,
        Ora, é preciso agüentar a tarefa.
        Trabalhei por um salário vil,
        Só para descobrir, assombrado,
