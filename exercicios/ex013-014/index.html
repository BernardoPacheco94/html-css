Capítulo 1

O melhor uso do CSS é dentro da tag <head> usando a tag <style> dentro do "cabeçalho". Dentro da tag <style> colocamos os elementos que queremos ajustar, como no exemplo: 

            body{
                background-color: aquamarine;
                font-family: Arial, Helvetica, sans-serif;
                font-size: 10px;
            }
            h1{
                color: darkblue;
                background-color: aqua;
            }
            h2{
                color: darkred;
            }
